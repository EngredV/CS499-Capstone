About This Project

This page will house the capstone project that represents the final enhancement of a mobile Android application originally built during the CS 360: Mobile Architecture and Programming course. It will be fully reimagined as a fullstack web application designed for Ornamaze, a handcrafted soap and skincare company. It will use modern technologies such as Next.js, FastAPI, and PostgreSQL.

Enhancement Categories

  • Software Engineering and Design: Will rebuild the system architecture by using fullstack frameworks and responsive UI.
  • Algorithms and Data Structures: Will implement intelligent filtering, alert logic, and custom sorting features.
  • Databases: Will migrate from SQLite to PostgreSQL and build a normalized schema with secure data handling.
